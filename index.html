Grid Gallery

Joomla! 3.x Module

Grid Gallery è un modulo Joomla che crea 4 differenti e fantastici layout di foto prese dagli album di phocagallery.

E' necessario avere installato sul proprio sito Joomla il componente Phocagallery.

  • HTML5/CSS3
  • Bootstrap 4
  • Font Awesome 5

Opzioni

  • 4 different layout
  • Container or Full container gallery
  • Random image option
  • Margin image set
  • Masonry or grid mode
  • Multi-album select
  • Image limit
  • Image overlay change color
